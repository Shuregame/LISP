18. Определите предикат, проверяющий, является ли аргумент одноуровневым списком.
```LISP
(defun spisok(lst)
      (cond
         ((null lst) "список одноуровневый")
          ((atom (car lst))(spisok(cdr lst)))
          ("список не одноуровневый")
       )
)

(print(spisok '((3))))
"список не одноуровневый" 

(print(spisok '(1 (5) ((9)))))
"список не одноуровневый" 

(print(spisok '(1 2 3)))
"список одноуровневый"
